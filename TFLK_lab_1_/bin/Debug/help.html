Рисунок 1 - Интерфейс программы


Пункт меню "Файл"

  • Создать - Создает новую вкладку в блокноте. Данная вкладка является чистой(без текста) и данный файл не хрониться в постоянной памяти.
  • Открыть - Открыть уже существующий файл с расширением .txt.
  • Сохранить - Сохранить все изменения в текущем файле, если файл является новым, то запускается проводник, где уже можно выбрать куда сохранить.
  • Сохранить как - Сохранить текущий файл в новое место.
  • Выход - Выход из программы

Правка

  • Отменить - Отменить сделанное действие, под действием понимается изменения текста.
  • Повторить -
  • Вырезать - Вырезать выделенный текст и поместить в системный буфер.
  • Копировать - Копировать выделенный текст в буфер.
  • Вставить - Вставить текст из буфера в окно ввода.
  • Удалить - Удалить выделенный текст.
  • Выделить все - Выделить весь текст в окне ввода.

Справка

  • Вызов справки - Вызвать справку о приложении.
